📖 使用说明

下载 exe

从上方下载 AutoMaskGenerator-v6.0.0-Portable.exe，放到任意目录。

双击运行

首次运行可能触发 Windows SmartScreen 提示，点击「更多信息 → 仍要运行」即可。

设置保存目录

底部状态栏显示默认保存目录（桌面），点击「📂 更改」按钮可切换到其他目录。此设置会被记忆。

正常使用

操作方式与 Web 版完全一致：导入图片 → 选择预设/手动吸色 → 分配通道 → 生成 Mask → 导出。

快速查看输出

菜单栏「文件 → 打开输出目录」或点击底部状态栏「📁 打开」，直接在资源管理器中查看导出文件。

🛠️ 开发者：自行构建

如果你需要自定义 Icon 或版本号，可以从源码自行构建：

# 1. 进入项目目录
cd tools/auto-mask-desktop

# 2. 安装依赖
npm install

# 3. 开发调试模式
npm start

# 4. 打包绿色免安装 exe
npm run build:portable

# 5. 打包标准安装包 (可选)
npm run build:nsis

自定义 Icon 步骤：

准备一个 256×256 的 PNG 图标

转换为 .ico：npx png-to-ico icon.png > build/icon.ico

重新执行 npm run build:portable
