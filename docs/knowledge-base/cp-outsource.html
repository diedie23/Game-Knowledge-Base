🏢 2. 供应商准入与分级体系

选 CP 就像选队友，能力是基础，但匹配度才是关键。最贵的不一定最好，最合适的才是最好。

— 采购PM十年心得

准入门槛：综合评分 ≥ 70 分方可入池。作品质量单项 < 60 分一票否决。

2.3 供应商四级分层

🏆 S级 — 战略合作伙伴

  • 定义：连续 3 次评估 90+ 分，质量稳定
  • 特权：优先派单、年框保底量、价格优惠
  • 管理：季度高层会面、联合培训
  • 占比：2~3 家，承担 40% 核心量

🥇 A级 — 主力供应商

  • 定义：评估 80~89 分，稳定可靠
  • 特权：正常派单、参与竞标
  • 管理：月度复盘、定期反馈
  • 占比：3~5 家，承担 35% 常规量

🥈 B级 — 备选供应商

  • 定义：评估 70~79 分，能力达标但需磨合
  • 特权：小批量试用、应急补位
  • 管理：重点辅导、每批次复盘
  • 占比：3~5 家，承担 20% 量

⚠️ C级 — 观察/淘汰

  • 定义：评估 60~69 分，或连续 2 次下滑
  • 处理：暂停派单、限期整改
  • 退出：整改后仍不达标则移出池
  • 占比：不超过总池 15%
🔴 制作人红线：S 级 CP 的认定必须制作人签字确认。核心资源（主角、主城、核心 UI）只允许 S/A 级 CP 承接。

💰 3. 采购谈判与合同策略

好的谈判不是压价，是找到双赢的结构。让 CP 有利润才有动力做好，但必须用结构绑定质量。

— 资深采购PM谈判心法

3.1 定价模式对比

模式适用场景优势风险PM建议
年框保底量产期、稳定需求单价低 15%~25%；CP 排期优先需求波动时浪费预算核心工种 S 级 CP 适用
项目制预研期、不确定需求灵活、无保底压力单价偏高、CP 优先级低新合作、试探期适用
阶梯报价量大可协商量越大越便宜需准确预估用量单工种月需求 >20 件时
竞标制大型单次项目充分竞争、价优管理成本高、关系维护难单次 >50 万项目时
人头驻场长期稳定需求管控强、灵活调度管理精力大、用工风险核心管线缺人时短期用

3.2 谈判策略工具箱

策略适用时机操作要点预期效果
批量折扣大批量需求承诺月均 N 件以上，换取单价下调 10%~15%成本降低，CP 有稳定收入
长约换价稳定合作方签 6~12 月长约，锁定价格 + 排期优先权双方都有确定性
质量奖金激励高品质一次通过率 > 90% 给予 5%~10% 奖金激励 CP 主动提升质量
阶梯惩罚约束延期延期 1~3 天扣 2%/天，>7 天扣 10%+重新评估CP 重视排期
备选竞压价格偏高时展示 2~3 家 CP 报价对比（脱敏后）自然降价 5%~15%
分阶付款大额合同首件30% + 中期30% + 终验40%降低财务风险
⚠️ 谈判底线原则：不能为压价牺牲质量。如果 CP 报价低于市场价 30%+，需警惕是否会用低价接单后降低品质或转包。

3.3 合同核心条款 Checklist（增强版）

📝 必备条款（缺一不可）

☐[交付物] 明确资源类型+数量+格式+规格+品质等级
☐[排期节点] 首件/中期/终验日期 + Buffer 天数
☐[验收标准] 引用验收 Checklist 作为合同附件
☐[修改次数] 上限 3 轮，超出另计费（单价的 20%/轮）
☐[付款方式] 分阶段付款比例 + 付款周期（验收后 N 天）
☐[原创声明] CP 保证资源非抄袭、非 AI 直出、非转包
☐[版权归属] 全部知识产权归属甲方
☐[保密条款] NDA 含泄露赔偿 + 竞品隔离
☐[延期违约] 延期扣款比例 + 严重延期终止权
☐[响应时效] 24 小时内必须回复，否则视为违约
☐[人员变更] 核心人员变更须提前 7 天书面通知
☐[争议解决] 约定仲裁/诉讼管辖地

🆕 v2.0 新增条款（采购PM视角）

☐[转包禁止] 未经甲方书面同意不得转包，违规立即终止+赔偿
☐[竞品隔离] 合作期内同品类竞品项目不得同时承接
☐[工具约束] 禁止使用 AI 生成工具作为最终交付，可作辅助但须申报
☐[质量奖惩] 一次通过率 > 90% 奖金条款 / < 60% 扣款条款
☐[审计权] 甲方有权审计制作过程（含工作环境、数据安全）
☐[退出机制] 连续 2 个评估周期不达标自动终止，无需赔偿
☐[涨价条款] 年度涨幅上限约定（通常 ≤5%/年）

4.3 首件审核制度（制作人视角增强版）

🔴 首件制度是外包质量的生命线：80% 的批量返工源于首件审核不严格。首件是整个批次的品质锚点，决定后续 100 件的方向。

审核维度审核人审核标准否决权
技术规范APM / TA面数/UV/贴图/命名全部达标任一 P0 不通过即打回
风格品质Art Lead / 主美与标杆一致、风格统一主美一票否决权
引擎表现TA / 程序引擎内实际效果、性能达标性能不达标即打回
制作人抽查制作人是否达到产品品质标准制作人一票否决权
🎬 制作人首件审核要点：
· 这个品质放到最终产品里，玩家会不会觉得"差"？
· 与竞品同类资源对比，我们是否有品质优势？
· 如果这个标准批量化，整体品质是否可接受？
· 这个 CP 的"天花板"在哪里？是否值得长期投入？

📊 5. 制作跟进与进度管控

外包跟进的核心不是"催"，而是"防"。等问题暴露再处理已经晚了，要在问题发生前设卡拦截。

— 采购PM进度管控心法

5.1 五级检查点体系

检查点时间检查内容形式异常处理
T1 首件审核启动后 3~5 天完成 1 件成品，全面评审在线会议 + 标注截图不通过重做 / 换人
T2 周进度同步每周固定时间完成量、在制量、问题清单进度表 + 截图连续 2 周滞后触发黄灯
T3 中期检查50% 进度时批量质量抽检、风格一致性抽查 3~5 件 + 报告质量偏差触发橙灯
T4 交付前预检交付前 2~3 天提前抽检最终成品在线预览重大问题延期交付
T5 正式交付约定日期全量检查，出具验收报告Checklist + 报告不通过退回 + 扣款评估

🎬 12. 制作人红线与品质底线

外包的品质底线不是"能用就行"，而是"放到产品里不会让玩家觉得差"。制作人的标准就是玩家的标准。

— 制作人品质哲学

12.1 制作人在外包管理中的角色定位

定位描述介入频率
品质守门人定义"什么品质是可接受的"底线首件+关键节点
方向决策者当外包方向有分歧时做最终裁决需要时
战略审批者S级CP认定、大额合同审批季度
危机处理者红灯事件的最终决策（终止/更换/坚持）偶发
🔴 制作人不应该做的事：
· 不参与日常进度跟催（那是 APM 的工作）
· 不直接和 CP 谈价格（那是采购 PM 的工作）
· 不逐件审核（只抽查关键件）
· 不频繁改变方向（改方向的成本 = 已完成量 × 单价）

12.3 制作人品质评级标准

品质等级定义对应产品场景典型标准
SSS 超品质超越竞品，可作为宣传卖点PV/宣传/核心角色高于竞品 Top10% 品质
S 高品质与竞品持平或略优主要角色/核心场景达到竞品平均品质
A 达标品质不拉低产品整体水平次要角色/普通场景玩家无感知差异
B 可用品质功能可用但品质一般临时/测试/内部验证仅限非正式场景
C 不可接受品质明显低于标准退回重做任何正式场景均不可用
🎬 制作人决策锦囊：
· 首件审核问自己三个问题：① 这个品质放到竞品里什么水平？② 批量化后整体观感如何？③ 玩家第一眼会不会觉得"廉价"？
· 当品质和排期冲突时：核心资源选品质，非核心资源选排期。永远不要让核心资源品质妥协。
· 当预算和品质冲突时：减量不减质。宁可少做 30% 数量，也不要 100% 都是平庸品质。
· 当 CP 达到天花板时：明确告知期望，给改进机会。如果连续 2 次仍不达标，友好更换不犹豫。

💡 附录：APM 外包管理核心心法

A.1 六字诀：清晰、前置、留痕

外包管理的底层逻辑：
· 清晰 — 需求越清晰，返工越少。模糊需求 = 返工成本 x 3
· 前置 — 首件制度、规范前置、风险前置。后期补救成本是前期预防的 5~10 倍
· 留痕 — 所有关键决策通过邮件确认。口头约定 = 没有约定

A.2 Don't vs Do 对照

❌ Don't — 常见误区

· "CP是专业的，不用说太细" → 每家标准不同，不说清就各做各的

· "首件差不多就行" → 批量返工成本是首件修改的 10x

· "群里说了就算数" → 事后扯皮无据可查

· "只看最终效果" → 到最后才发现问题已无力回天

· "最便宜的更省钱" → 低价CP返工率高，综合成本反而贵

· "一家CP做所有" → 单点依赖风险巨大

· "压到最低价" → CP没利润就没动力，品质必然下滑

· "出问题再说" → 等暴露已是最贵的时间修复

✅ Do — 正确做法

· 提供详细规范 + 参考图 + 标杆包 + 交付模板

· 首件必须严格达标后才批量，制作人把关

· 重要决定邮件确认，关键对话截图备份

· 设置 5 级检查点 + 红黄绿灯预警

· 用 CP 评分卡科学选择，数据说话

· 核心资源至少 2 家 CP 备选

· 用结构（奖惩/阶梯/长约）替代单纯压价

· 数据驱动：追踪趋势，提前干预

A.3 制作人金句 — 外包管理篇

🎬 关于品质

· "外包品质的上限取决于你的标杆包，下限取决于你的红线清晰度"

· "不要指望 CP 超水平发挥，他们的天花板就是首件的品质"

· "验收时问自己：如果竞品看到这个品质，会嘲笑还是紧张？"

💰 关于成本

· "最贵的成本不是 CP 单价，而是方向错误后的全量重做"

· "省钱的最佳方式不是砍价，而是减少返工次数"

· "一个好的首件审核，胜过十轮批量修改"

🤝 关于关系

· "把 CP 当合作伙伴而不是供应商，他们会多给你 20% 的用心"

· "但合同条款一个字都不能松，这是保护双方的底线"

· "行业圈子很小，每次合作都是在建立口碑"

⚡ 关于效率

· "外包管理 80% 的功夫在需求下发前，20% 在下发后"

· "如果你每天都在催进度，说明你的检查点设得不够"

· "好的外包管理是让 CP 知道你比他更清楚进度"
