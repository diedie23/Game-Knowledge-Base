|:---:|:---:|:---:|:---:|:---:|
| 初级美术 (1~3年) | ¥12,000 | ¥16,800 | ¥20,160 | **¥20,000** |
| 中级美术 (3~5年) | ¥20,000 | ¥28,000 | ¥33,600 | **¥34,000** |
| 高级美术 (5~8年) | ¥30,000 | ¥42,000 | ¥50,400 | **¥50,000** |
| 资深/主美 (8年+) | ¥45,000 | ¥63,000 | ¥75,600 | **¥76,000** |

## 2️⃣ 工种系数矩阵

### 📊 2.1 不同工种的效率系数

> **[核心]** 系数反映了该工种的技术门槛和人才稀缺度。

| 🎨 工种 | 📊 系数 | 📖 含义 | 📝 说明 |
|:---:|:---:|:---:|:---:|
| **角色原画** | 1.2 | 高于基准 | 创意性强，修改成本高 |
| **角色建模** | 1.0 | 基准 | 标准化程度适中 |
| **场景建模** | 0.9 | 略低 | 模块化程度高，可复用 |
| **UI 设计** | 0.8 | 较低 | 规范化程度高 |
| **特效制作** | 1.3 | 最高 | 技术门槛高，人才稀缺 |
| **动画制作** | 1.1 | 高于基准 | 需要专业技能 |
| **TA (技术美术)** | 1.5 | 最高 | 复合型人才溢价 |

### 🧮 2.2 实际成本计算

> 💡 **计算公式**：
>
> `工种实际人月成本 = 基线人月成本 × 工种系数`
>
> **[示范]**：中级美术做特效 = **¥34,000 × 1.3 = ¥44,200/月**

## 3️⃣ ROI 评估模板

### ⚖️ 3.1 外包 vs 内部 ROI 对比

> **[决策参考]** 六维度对比帮助快速判断外包还是内部。

| 📊 维度 | 🏢 内部 | 🏭 外包 | 🏆 谁更优 |
|:---:|:---:|:---:|:---:|
| **单位成本** | ¥34,000/月 (中级) | ¥22,000/月 (B级CP) | 外包 ✅ |
| **质量可控性** | 高 | 中~低 | 内部 ✅ |
| **沟通成本** | 低 | 中~高 | 内部 ✅ |
| **灵活性** | 低（固定人数） | 高（弹性扩缩） | 外包 ✅ |
| **知识沉淀** | 高 | 低 | 内部 ✅ |
| **风险** | 低 | 中（质量/交付） | 内部 ✅ |

### 🧮 3.2 外包 ROI 计算公式

> 💡 **公式**：
>
> `外包ROI = (内部等价成本 - 外包实际成本 - 管理成本) / 外包实际成本 × 100%`

**[示范]**：5 个 A 级角色建模，内部需 3 名中级美术做 2 个月

| 📌 项目 | 💰 金额 |
|:---:|:---:|
| 内部等价成本 | 3人 × ¥34,000 × 2月 = **¥204,000** |
| 外包实际成本 | **¥175,000**（含修改） |
| 管理成本 | ¥8,000（APM 跟进 0.2 人月） |
| **ROI** | **(204,000 - 175,000 - 8,000) / 175,000 = 12%** |

### 🧭 3.3 外包决策矩阵

> **[决策参考]** 四象限决策法——内部产能 × 资产核心度。

| 📋 条件 | 💡 建议 |
|:---:|:---:|
| 内部产能充足 + 核心资产 | 🟢 内部做 |
| 内部产能充足 + 非核心资产 | 🟡 可内部或外包 |
| 内部产能不足 + 核心资产 | 🟠 外包 + 高标准验收 |
| 内部产能不足 + 非核心资产 | 🟢 外包 |
| 技术门槛高（TA/特效） | 🟠 优先内部，外包需S级CP |

## 5️⃣ 成本优化策略

### 📉 5.1 降本增效手段

| 💡 策略 | 📊 预期效果 | 🎯 实施难度 |
|:---:|:---:|:---:|
| **建立资产复用库** | 减少 15~20% 重复制作 | 中 |
| **标准化模板** | 减少沟通和返工 | 低 |
| **工具自动化** | 减少 10~15% 人力 | 中 |
| **优化外包结构** | 核心自做，非核心外包 | 低 |
| **培养内部 TA** | 减少外部 TA 采购 | 高 |

> ⚡ **APM 金句**：成本优化不是砍预算，而是让每一分钱都花在刀刃上。
