📋 检测等级说明

等级含义处理方式
✅ 通过所有规则检测通过可直接入库
⚠️ 警告非致命问题（如空 Alpha、文件较大）建议优化后入库
❌ 不通过关键规则违反（尺寸超标、格式错误）必须修正后重新提交

⚠️ 注意事项

  • TGA 文件通过解析 18 字节 Header 获取尺寸/Alpha 信息，支持完整规则检测
  • PSD 文件通过解析文件头获取尺寸/通道数/色彩模式，支持完整规则检测
  • 色彩空间检测基于 PNG sRGB chunk / iCCP Profile / gAMA 值解析，非 PNG 格式暂标记为"未知"
  • 批量检测采用多路并行加速（自适应 CPU 核心数），500 个文件检测速度提升 3-8 倍
  • 文件在浏览器本地处理，不会上传到任何服务器
