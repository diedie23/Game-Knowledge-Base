⚖️ 二、核心决策矩阵（增强版）

📦 适合外包

  • 批量标准化资产 — NPC、杂兵、道具等低创意性
  • 工艺成熟度高 — 有明确规范+可量化验收标准
  • 内部产能饱和 — 利用率>85%持续4周+
  • 非核心IP — 不涉及品牌形象/核心设定
  • 短期峰值 — <6月临时产能补充
  • 特殊工艺 — 内部不具备的技能（动捕后处理等）
  • 可并行分包 — 工作包间低耦合
  • 市场标杆充足 — 同类资产大量参考，低沟通歧义

🏠 适合内发

  • 核心IP角色 — 主角、品牌形象、宣传物料
  • 高创意性 — 原画概设、风格探索、视觉开发
  • 高频迭代 — 日频以上修改的实验性内容
  • 机密性 — 未公开核心设定
  • 能力建设 — 沉淀核心技术、培养梯队
  • 强耦合 — 与程序/策划高频联调
  • 品质天花板 — 名片级品质零容忍
  • 长期维护 — 上线后持续迭代的系统性资产

🤝 适合混合模式

  • 分层制作 — 外包白模/中模，内部高精度+材质
  • UI/UX系统 — 标准组件外包，交互原型内发
  • 大型场景 — 模块化组件外包，组装灯光内发
  • 动画系统 — 基础动画外包，表情关键动画内发
  • 特效系统 — 通用粒子外包，技能特效内发

🔍 灰色地带（需深入分析）

  • B级角色（有剧情）— 评估曝光频次和商业价值
  • 新工艺首次尝试 — 学习成本 vs 外部成熟经验
  • PV/过场动画 — 外部导演资源 vs 内部效率
  • 技术美术需求 — 工具属性 vs 资产属性
  • 赛季皮肤 — 概设内发+制作可外包
⚠️ PM核心认知：正确决策 = 全链路TCO + 质量风险 + 时间效率 + 战略价值的四维评估。外包隐性成本常被低估30-50%，内发闲置浪费同样容易被忽视。

3.1 加权评分公式

📐 决策得分：Δ = Σ(外包分×权重) - Σ(内发分×权重)
Δ>+0.5→外包 | Δ∈[-0.5,+0.5]→混合 | Δ<-0.5→内发

💰 四、全链路成本对比模型

4.1 外包TCO公式

📐 外包TCO = 合同价×(1+返修系数) + 管理成本 + 沟通成本 + 风险准备金 + 机会成本
· 返修系数：首次CP=0.3~0.5 | S级CP=0.05~0.15
· 管理≈12~18% | 沟通≈8~15% | 风险≈5~15% | 机会=PM时间×时薪

4.2 内发TCO公式

📐 内发TCO = 人天×日薪×(1+管理系数) + 固定成本 + 培训成本
· 初级¥1,000~1,500 | 中级¥1,500~2,500 | 高级¥2,500~4,000/天
· 管理≈5~10% | 固定≈¥250~400/天 | 新人前3月产能折损30~50%

4.4 隐性成本陷阱

🚨 外包隐性成本

  • 需求文档：比内部详尽3~5倍(≈8~12h/批次)
  • 版本冲突：多CP并行的文件管理开销
  • 验收会议：每周2~4h评审会(美术TL+PM)
  • 风格校准：CP换人需重新对齐(≈3~5天)
  • 行政结算：月度对账、发票、审批流
  • 返工阻塞：等待返修期间其他环节的时间成本

🚨 内发隐性成本

  • 闲置浪费：空窗期利用率≈60~70%
  • 招聘周期：HC到到岗2~3月
  • 新人培养：前3月产能≈50%，前6月≈80%
  • 离职断层：关键人员流失的知识损失
  • 加班隐损：长期高压→效率↓+流失风险↑
  • 技能退化：长期同类项目→能力天花板

8.3 CP池管理

✅ 最佳实践

  • 2+1法则：每工种2家成熟+1家备选
  • 分级分包：S/A承接高价值，B做标准批量
  • 梯队培养：每季度给C级1-2批试水
  • 季度复盘：数据驱动优胜劣汰
  • 关系维护：S级给予排期/价格优先

❌ 常见错误

  • 过度依赖单CP（>60%）→一损俱损
  • 仅凭报价选CP → 低价陷阱
  • 不试水直接放量 → 首批翻车
  • 无退出机制 → 僵尸CP耗精力
  • 不维护关系 → 旺季被抢走产能

📈 十二、ROI量化模型与决策复盘

12.1 外包ROI公式

📐 外包ROI = (内发TCO - 外包TCO) / 外包TCO × 100%
示例A：B级NPC → ROI=(5200-4050)/4050=+28.4%（外包划算）
示例B：S级主角 → ROI=(18000-22500)/22500=-20%（内发划算）

💎 十五、金句、原则与检查清单

"外包不是偷懒，内发不是浪费。正确的决策是让合适的人做合适的事，在合适的时间以合适的成本交付合适的品质。"

❌ 决策反模式

  • "所有都外包省钱" — 隐性成本吃掉利润
  • "核心也外包有验收就行" — 风格失控迟早的事
  • "紧急就一定外包" — 紧急+首次=翻车
  • "拍脑袋决策" — 感性必留遗患
  • "永远不外包" — 产能瓶颈拖垮项目
  • "出问题全转内发" — 可能只是CP选错

✅ 决策金律

  • 标准化外包，创意性内发
  • 核心IP绝不外包
  • 首次CP先小批量试水
  • 决策有数据+流程+签字
  • 混合模式边界清晰
  • 季度复盘持续优化

决策前Checklist

✅ 提交决策报告前确认：
□ 红线检查通过，无一票否决 □ 十维度评分完成，结论明确 □ TCO对比含所有隐性成本 □ 风险评估完成+Plan B □ CP选型有数据支撑 □ 合同关键条款已审核 □ 混合边界定义清楚 □ 制作人已口头认可
