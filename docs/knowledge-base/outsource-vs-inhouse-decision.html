⚖️ 二、核心决策矩阵

📦 适合外包的情况

  • 批量标准化资产——NPC、杂兵、场景道具等低创意性工作
  • 工艺成熟度高——有明确规范和验收标准
  • 内部产能饱和——核心团队已满负荷运转
  • 非核心 IP 资产——不涉及核心角色设定
  • 短期峰值需求——量产冲刺期的临时产能补充
  • 特殊工艺补充——内部团队不具备的技能

🏠 适合内发的情况

  • 核心 IP 角色——主角、重要 NPC、品牌形象
  • 高创意性工作——原画概设、风格探索、视觉开发
  • 迭代频率高——需要频繁修改的实验性内容
  • 机密性要求——预研期未公开的核心设定
  • 长期能力建设——需要沉淀团队核心技术
  • 跨部门强耦合——需要与程序/策划高频联调
⚠️ 常见误区：不要仅因为"省钱"就选择外包。外包的隐性成本（沟通成本、返修成本、管理成本）经常被低估 30-50%。真正的决策应基于全链路成本 + 质量风险 + 时间效率的三角评估。

📊 三、七维度评估体系

对每个维度进行 1-5 分打分，加权计算后得出总分。外包得分 > 内发得分 → 建议外包，反之建议内发。

💰 四、成本对比模型

4.1 外包全链路成本公式

📐 外包真实成本 = 合同价 × (1 + 返修系数) + 管理成本 + 沟通成本 + 风险准备金

其中：
· 返修系数：首次合作 CP = 0.3~0.5 | 成熟 CP = 0.1~0.2
· 管理成本 ≈ 合同价 × 15%（APM 管理工时折算）
· 沟通成本 ≈ 合同价 × 10%（跨团队沟通、验收评审）
· 风险准备金 ≈ 合同价 × 10%（延期、返工的缓冲）

4.2 内发全链路成本公式

📐 内发真实成本 = 人天 × 日薪成本 × (1 + 管理系数) + 工位固定成本

其中：
· 日薪成本：内部美术约 ¥1,200~2,500/天（含五险一金、福利）
· 管理系数 ≈ 0.05~0.1（内部沟通成本低）
· 工位固定成本 ≈ ¥300/天（场地、设备折旧）

⚠️ 五、风险对比分析

📦 外包风险清单

  • 🔴 质量波动大——CP 内部人员变动导致风格不一致
  • 🔴 返修周期长——远程沟通增加 2-3 天返修周期
  • 🟡 泄密隐患——资产源文件外流风险
  • 🟡 交付延期——CP 同时接多个甲方，优先级不可控
  • 🟡 沟通损耗——需求理解偏差导致首版通过率低
  • 🔵 供应商依赖——核心 CP 被锁定，替换成本高

🏠 内发风险清单

  • 🔴 人力刚性——短期无法扩充，峰值需求无法满足
  • 🟡 团队倦怠——长期高负荷加班导致流失
  • 🟡 单点风险——关键人员离职造成交付断裂
  • 🔵 成本刚性——固定人力成本无法随项目周期弹性调整
  • 🔵 技能瓶颈——缺乏特定工艺导致质量天花板
🚨 致命风险场景：「首次合作 CP + S 级核心角色 + 紧急工期」—— 此组合返修率高达 60%+，几乎必然翻车。绝对不要在这种情况下选择外包。

💎 八、金句与原则

"外包不是偷懒，内发不是浪费。正确的决策是让合适的人做合适的事，在合适的时间以合适的成本交付。"

❌ 决策反模式

  • "所有资产都外包，省钱第一"——隐性成本会吃掉利润
  • "核心角色也外包，反正有验收"——风格失控是迟早的事
  • "紧急就一定要外包"——紧急 + 首次合作 = 翻车
  • "拍脑袋决策，不做成本分析"——感性决策必留遗患

✅ 决策金律

  • 标准化外包，创意性内发——明确分工边界
  • 核心 IP 绝不外包——这是团队的命根子
  • 首次合作 CP 先小批量试水——验证后再放量
  • 决策有数据、有流程、有签字——可追溯可问责
