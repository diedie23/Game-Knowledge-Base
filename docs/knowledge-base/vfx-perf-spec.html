🎯 2. 特效层级管理规范

2.1 渲染层级定义
渲染顺序（从后到前）：Layer 0 场景底层 → Layer 1 场景环境 → Layer 2 角色底层 → Layer 3 角色身体 → Layer 4 角色技能 → Layer 5 命中 → Layer 6 前景 → Layer 7 UI
层级名称示例
Layer 0场景底层特效地面裂缝/水面波纹
Layer 1场景环境特效落叶/雨雪/雾气
Layer 2角色底层特效脚底光环/站立印记
Layer 3角色身体特效Buff/Aura/附魔
Layer 4角色技能特效释放/飞行/爆炸
Layer 5命中特效Hit/受击反馈
Layer 6前景特效全屏闪白/镜头震动
Layer 7UI 特效按钮光效/升级动画
2.2 层级优先级与裁剪策略
层级优先级低端机裁剪策略
Layer 7: UI 特效⭐⭐⭐⭐⭐不裁剪，但简化粒子数
Layer 4: 技能特效⭐⭐⭐⭐⭐降低粒子数 50%，贴图降级
Layer 5: 命中特效⭐⭐⭐⭐缩短生命周期

Layer 0: 底层特效⭐可完全关闭

🚨 5. 常见性能雷区

以下是特效制作中最常见的性能陷阱，每一项都可能导致帧率暴跌。
5.1 🔴 Alpha 叠加过多
项目说明
现象多层半透明粒子叠加导致 Overdraw 飙升
案例火焰特效用 20 层半透明粒子叠加，Overdraw 达 15x+
解法减少粒子层数，用贴图细节代替；使用 Additive 混合代替 Alpha Blend；远距离自动降低粒子数
5.2 🔴 大面积半透明
项目说明
现象全屏大面积半透明面片导致 GPU 填充率爆炸
解法限制半透明面片最大不超过屏幕 1/4；用 Mask 裁切；全屏效果优先用 Post-Process
5.3 🟡 未合批的粒子系统
项目说明
现象相同材质的粒子未合批，Drawcall 翻倍
解法相同材质粒子使用同一发射器；启用 GPU Instancing；合并贴图到 Atlas
5.4 🟡 不必要的灯光交互
项目说明
现象特效粒子接受场景灯光影响，增加 Shader 开销
解法特效默认设置为 Unlit；仅特殊需求下开启灯光影响
5.5 🟡 序列帧贴图过大
项目说明
现象64 帧序列帧打成 4096×4096 的 Atlas
解法控制序列帧总帧数 ≤ 16 帧；单帧 ≤ 64×64；Atlas ≤ 512×512

🚨 7. 典型问题

7.1 新特效一上线帧率立即暴跌
🚨 案例 1：新特效上线帧率暴跌——战斗 FPS 从 45 降至 18 高频

🎬 典型场景还原

新版本上线后，战斗场景 FPS 从 45 降至 18，定位到是新增的大招特效导致。

🔍 问题根因拆解

• 特效未经中端机性能测试即提交，开发机性能掩盖了问题

• 特效使用了 30 层半透明粒子叠加，Overdraw 达到 15x

• 未制作 LOD 版本，低端机使用了与高端机相同的效果

💡 APM 破局思路

• 紧急 hotfix：将该特效临时替换为 LOD2 简化版

• 重新制作 3 级 LOD 版本，中端机 Overdraw 控制在 4x 以内

• 所有特效补充「性能报告卡」，经 TA Review 后才能提交

• 严格执行第 6 章优化 Checklist，必须在目标低端机上真机测试通过

• 在 CI/CD 中加入 Drawcall / 粒子数自动检测，超标自动阻断提交

7.2 低端机全白屏闪烁
🚨 案例 2：低端机全白屏闪烁——大招释放时 GPU Stall

🎬 典型场景还原

低端机上释放大招时出现约 0.5 秒的全白屏闪烁，用户体验极差。

🔍 问题根因拆解

• 大招特效使用了全屏半透明面片做「闪白」效果

• 低端机 GPU 填充率瞬间超出预算，产生 GPU Stall

• 层级管理缺失，全屏效果（Layer 6）未做低端机适配

💡 APM 破局思路

• 全屏闪白效果改用 Post-Process 实现，避免半透明面片

• 低端机上关闭 Layer 6 前景特效或替换为极简版

• 全屏效果必须用 Post-Process，禁止使用全屏半透明面片

• 每个层级的特效都需要低端机适配版本

• 真机测试时重点关注大招/连招场景

📌 8. Do / Don't 示例

场景说明：特效师为新英雄制作技能特效，需在中端机上达到性能红线标准。

✅ Do（正确示范）

• 制作前查阅本文档红线指标，确认中端机预算：DC ≤ 8、粒子 ≤ 200、Overdraw ≤ 4x

• 使用通道复用将 4 张贴图合并为 1 张，贴图内存控制在 1.5MB

• 制作 LOD0/LOD1/LOD2 三个版本，低端机使用 LOD2（粒子 25%、贴图降两级）

• 本地 Profiler 自测通过后填写性能报告卡，提交 TA Review

• 在目标低端机（iPhone 11）上实际测试 10 分钟战斗，帧率 ≥ 30 FPS

❌ Don't（错误示范）

• 在高端开发机上调好效果直接提交，不做任何性能测试

• 用 20 层半透明粒子堆效果，「看起来好看就行」

• 只做一个版本，不制作 LOD，低端机也跑满配效果

• 全屏大招效果用全屏半透明面片，不用 Post-Process

• 不填写性能报告卡，不经过 TA Review 直接入库
