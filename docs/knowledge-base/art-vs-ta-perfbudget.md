# 性能红线与资产预算

> **适用阶段**：预研期 | **优先级**：高 | **负责人**：孙七
>
> 本文档定义模型面数、贴图尺寸、骨骼数、Drawcall 预算表，按品类与目标机型分级的资产红线标准。

---

## 1. 机型分档策略

### 1.1 目标机型分档

| 档位 | 代表设备 | 市场份额 | 优先级 |
|------|---------|---------|--------|
| 🔴 **高端** | iPhone 15 Pro / 骁龙 8 Gen3 / 天玑 9300 | ~20% | 画质标杆 |
| 🟡 **中端**（基准档） | iPhone 13 / 骁龙 870 / 天玑 8200 | ~50% | **核心优化目标** |
| 🟢 **低端** | iPhone 11 / 骁龙 765G / 天玑 700 | ~30% | 保证可玩 |

> ⚠️ **黄金法则**：以中端机为基准设定红线，高端机加特效，低端机裁剪。

### 1.2 帧率目标

| 场景 | 高端 | 中端 | 低端 |
|------|------|------|------|
| 大厅/主城 | 60 FPS | 30 FPS | 30 FPS |
| 战斗（同屏 5 角色） | 60 FPS | 30 FPS | 30 FPS |

## 5. Drawcall 预算

### 5.1 全局 Drawcall 预算（中端）

| 类别 | 预算 |
|------|------|
| 角色（同屏 5 个） | ≤ 50 DC |
| 场景 | ≤ 150 DC |
| 特效 | ≤ 50 DC |
| UI | ≤ 30 DC |
| 阴影 | ≤ 30 DC |
| 其他（天空/后处理等） | ≤ 20 DC |
| **总计** | **≤ 330 DC** |

### 5.2 单资产 Drawcall 红线

| 资产 | 红线 |
|------|------|
| 单个主角 | ≤ 5 DC（Body + Face + Weapon + Hair） |
| 单个 NPC | ≤ 3 DC |
| 单个特效 | ≤ 8 DC |
| UI 单界面 | ≤ 10 DC（合理使用 Atlas） |

### 5.3 降低 Drawcall 的方法

| 方法 | 说明 | 适用 |
|------|------|------|
| **材质合并** | 多个部件共用一个材质 | 角色/场景 |
| **贴图 Atlas** | 多张小图合并为大图集 | UI/特效 |
| **Static Batching** | 静态合批 | 场景 |
| **Dynamic Batching** | 动态合批（小网格） | 场景道具 |
| **GPU Instancing** | 相同网格实例化 | 植被/粒子 |
| **SRP Batcher** | Unity SRP 自动合批 | 全局 |
| **Merge Actor** | UE 合并 Actor | 场景 |
