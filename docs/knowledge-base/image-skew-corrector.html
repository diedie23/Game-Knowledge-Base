📖 使用说明

操作步骤
  1. 添加文件 — 点击"添加文件"或"添加文件夹"按钮，或将文件拖拽到窗口中
  2. 设置参数 — 调整角度阈值、输出模式、背景填充色
  3. 开始处理 — 点击"开始处理"按钮
  4. 查看日志 — 底部日志区域显示处理进度和结果
检测算法说明
算法原理适用场景
霍夫变换检测图像中的直线，计算主要方向有明显线条的图片
最小面积外接矩形找到包围内容的最小矩形，计算旋转角规则形状内容
PCA 主轴分析通过主成分分析确定内容方向不规则分布内容
投影方差法尝试不同角度，找到投影方差最大的方向文本类图片
