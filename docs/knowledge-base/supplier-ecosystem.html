🌐 1. 供应商生态管理总览

核心理念：美术外包不应只停留在"一单一验"的项目级交易，而应构建长期、可持续、分级分层的供应商生态。好的生态管理让顶级 CP 愿意留下来、新 CP 能快速成长、差 CP 被有序淘汰。

供应商生态管理涵盖从寻源准入 → 试用评估 → 正式合作 → 定期评级 → 升降级 → 淘汰退出的全生命周期闭环。

📊 供应商分级总览
等级标识定义合作权限结算优惠
S 级 · 战略S连续 2 次年度评级 A+ 的核心供应商优先下单 + 项目驻场 + 联合研发月结 T+15，可预付 30%
A 级 · 优选A综合评分 ≥ 85 且无安全事故批量下单 + 中大型项目月结 T+30
B 级 · 合格B综合评分 60~84小中型项目 + 限制单批量上限验收后 T+45
C 级 · 观察C综合评分 40~59，或连续 2 期 B-仅限小型试制 + 必须加中期检查验收后 T+60
D 级 · 淘汰D综合评分 < 40 或触发黑名单规则停止合作 + 结算遗留 + 列入黑名单结清后终止

🔴 铁律：所有新 CP 进入生态时默认为 B 级（合格），需通过试制期后根据实际表现调整等级。任何 CP 不可跳过试制期直接升至 A/S 级。

🔍 2. CP 寻源标准与准入机制

📡 2.1 寻源渠道
渠道优势劣势适用场景
行业推荐质量有保障，口碑可查圈子有限核心角色/高品质需求
外包平台（米画师/CG 模型网等）候选量大，比价容易质量参差不齐大批量 + 标准化资产
展会/比赛（ChinaJoy/GGAC）发现潜力新团队转化周期长储备新 CP
内部推荐了解对方实力关系偏差快速补充
CP 自荐有合作意愿需严格筛选长期储备池
📋 2.2 准入评估维度

🔍 CP 准入评估 Checklist（APM 填写）

☐公司资质：营业执照 + 无失信记录 + 成立 ≥ 1 年
☐团队规模：核心美术 ≥ 3 人，有稳定产能
☐作品匹配度：Portfolio 与项目风格匹配度 ≥ 70%
☐工程规范意识：了解命名/尺寸/格式等技术要求的重要性
☐沟通效率：初次沟通响应 ≤ 24h，理解需求准确
☐安全合规：愿意签署 NDA + 使用指定传输渠道
☐报价合理：报价在行业中位数 ± 20% 范围内
☐试制意愿：愿意接受 2~3 个资产的有偿试制评估

📊 2.4 试制评分标准
维度权重评分标准（1~10 分）不及格线
视觉品质30%与原画一致性、细节完成度、风格匹配度< 5 分
工程合规25%命名/尺寸/格式/通道全部正确< 6 分
交付准时20%在约定时间内交付，无延期延期 > 2 天
沟通效率15%理解需求、响应速度、修改配合度< 4 分
性价比10%品质/价格比，与同类 CP 横向对比< 4 分
✅ 准入标准：试制综合加权分 ≥ 60 分且无任何维度不及格 → 准入 B 级。综合分 ≥ 80 分 → 快速通道，首期可下发中型批次。

📊 3. 半年度 / 年度评级体系

核心原则：供应商评级不是"拍脑袋"，而是数据驱动 + 多维度加权。每半年一次正式评级，评级结果直接决定下半年的合作权限和结算条件。
📐 3.1 评级维度与权重
维度权重核心指标数据来源
资产质量35%一次通过率、平均返工轮次、主美评分评审系统 + JIRA
交付效率25%准时交付率、响应速度、产能稳定性项目管理工具
成本表现20%实际 vs 报价偏差、ROI、隐性成本（沟通+管理）财务系统 + APM 记录
安全合规10%NDA 执行情况、文件传输合规率、0 泄密事件安全审计
合作态度10%主动沟通频率、问题修复积极性、改进意愿APM 主观评分
📊 3.2 评级计算公式
📊 供应商综合评分 (Supplier Score, SS)
SS = Σ(维度得分 × 维度权重) = Q×35% + D×25% + C×20% + S×10% + A×10%

Q = 质量得分（0~100）| D = 交付得分 | C = 成本得分 | S = 安全得分 | A = 态度得分

每个维度采用百分制，最终 SS 也为百分制。

📋 3.3 评级等级映射
综合评分等级含义下半年合作权限
≥ 90A+卓越 — 战略级候选优先扩大份额 + 联合项目 + 快速结算
85~89A优秀批量下单 + 中大型项目
75~84B+良好中型项目 + 适度扩量
60~74B合格维持现有份额
40~59C观察 — 需改进缩减份额 + 加强验收 + 限期整改
< 40D不合格 — 淘汰停止新单 + 结算遗留 + 列入黑名单

📈📉 4. 升降级与淘汰机制

📈 4.1 升级条件
升级路径条件额外要求
B → A连续 2 次半年评级 ≥ 85 分无安全事故 + 无 P0 级质量问题
A → S（战略）连续 2 次年度评级 A+ 且合作 ≥ 2 年制作人审批 + 签署战略合作协议
C → B（恢复）整改期内评分提升至 ≥ 60 分APM 出具整改验证报告
📉 4.2 降级条件
降级路径触发条件处理措施
A → B单次评级 < 75 分缩减份额 + APM 约谈
B → C单次评级 < 60 分 或 连续 2 次 B-限制为小型试制 + 限期整改 3 个月
C → D整改期未达标 或 发生安全事故停止合作 + 列入黑名单
一票否决重大泄密 / 伪造资产 / 商业欺诈立即终止 + 永久黑名单 + 法律追究

🔴 铁律：一票否决不受评级周期限制，任何时间发现即触发。涉及知识产权侵权（使用盗版素材/AI 生成资产冒充原创）同属一票否决范围。

🚫 5. 黑名单与限制合作制度

📋 5.1 黑名单触发条件
类型触发条件黑名单期限解除条件
永久黑名单重大泄密 / 伪造资产 / 盗用素材 / 商业欺诈永久不可解除

📋 5.2 黑名单管理流程

🚫 列入黑名单操作清单

☐书面通知：通过正式邮件通知 CP，说明原因和期限
☐系统标记：在供应商管理系统中标记为"黑名单"状态
☐账号冻结：关闭 CP 的资产传输通道、项目云盘权限
☐源文件回收：确认所有项目相关源文件已回收或销毁确认
☐内部通报：在内部供应商库中标注，防止其他项目组重复合作
☐法务备案：涉及永久黑名单的，同步法务部门备案
⚠️ 注意：黑名单 CP 的关联公司（同一法人/实际控制人名下的其他公司）也应被标记为"关联风险"，合作前需额外评估。

🔒 6. 资产安全合规与防泄密 SOP

核心原则：游戏美术资产是核心商业机密。泄密 = 竞品抢跑 + 玩家剧透 + 品牌损失。必须从合同、传输、存储、人员四个维度构建防线。
📄 6.1 合同层 — NDA 与知识产权
条款内容要求违约后果
NDA（保密协议）合作前必签；保密期限 ≥ 项目发布后 3 年违约金 + 法律追究
知识产权归属所有外包资产著作权完全归甲方所有侵权赔偿
竞业限制合作期内不得同时为直接竞品项目制作同类资产合同终止 + 违约金
人员清单CP 必须提供参与项目的人员名单，人员变动需报备未报备视为违规
📡 6.2 传输层 — 安全传输 SOP
环节✅ 允许❌ 禁止
文件传输加密云盘（企业微盘/OneDrive/自建 FTP over TLS）QQ 群文件、百度网盘分享链接、微信传文件
即时沟通企业微信（可审计）个人微信、QQ（不可追溯）
评审交流线上会议 + 水印截屏截屏后通过社交媒体外传
源文件存储甲方指定的加密磁盘 / 云空间CP 个人电脑本地留存（项目结束后必须删除）
🖥️ 6.3 存储层 — 资产水印与追溯

所有下发给 CP 的参考资料、原画设计稿必须添加：

水印类型内容位置
可见水印CP 名称 + 日期 + "机密"字样图片对角线半透明覆盖
隐形水印唯一追溯 ID（哈希值）嵌入图像数据
文档水印"内部资料 禁止外传 + CP 名称"每页页脚 + 背景浮水印

🤝 7. 供应商关系维护与赋能

理念：好的生态不是只"管"和"罚"，更要"帮"和"赋能"。帮助 CP 成长 = 帮助自己减少验收成本 + 提升资产品质。
📋 7.1 定期赋能措施
措施频率内容目标
风格培训会项目初期 + 风格变更时项目美术风格宣讲、参考图库共享、标杆资产展示减少风格偏差
技术规范同步每季度最新命名/尺寸/格式规范、自检工具更新包减少工程打回
1-on-1 反馈会每月（A/S 级） 每季（B 级）上期表现回顾、问题分析、改进建议持续优化
优秀资产展示每季度评选"最佳外包资产"，在内部 + CP 群展示正向激励
工具赋能持续向 CP 提供自检工具/脚本、模板文件、规范文档提升首次合规率
🎁 7.2 激励机制
激励类型条件内容
🏆 年度最佳 CP年度评级 A+ 且合作满 1 年荣誉证书 + 下年度报价上浮 5% 空间 + 优先分配
📈 份额奖励连续 2 次评级 A 级以上下期份额增加 20~30%
💰 快速结算S 级战略供应商月结 T+15（vs 普通 T+45）

🚨 8. 典型问题

🔴 案例："老关系" CP 品质下滑却无人敢动 高频

🎬 典型场景还原

主美："CP-X 这半年交付质量明显下降，一次通过率只有 35%。"

APM："可是 CP-X 是项目初期就合作的老关系，制作人那边不好说…"

制作人："他们之前还行吧，再给一次机会？"

结果：又过了半年，CP-X 继续恶化，期间 50+ 个资产需要返工，浪费了 ≈ 3 个内部人月的验收和沟通成本。

💡 破局思路

1. 用数据说话：评级体系量化，35% 的一次通过率 = C 级，自动触发限期整改
2. 制度代替人情：升降级规则提前公示，所有 CP 一视同仁
3. 设置缓冲期：不是立即淘汰，给 3 个月整改机会，但必须有明确目标
4. 准备替代方案：在降级的同时启动新 CP 寻源，避免"不敢淘汰因为没人替"

🔴 案例：外包资产疑似使用 AI 生成并冒充手绘 新兴

🎬 典型场景还原

主美："这批角色概念图看着不太对…手部细节诡异，光影有 AI 生成痕迹。"

APM："CP 说是手绘的…但确实和 Midjourney 出图风格很像。"

💡 破局思路

1. 合同中明确禁止未授权使用 AI 生成内容冒充原创
2. 要求 CP 提交关键资产的PSD/CSP 图层文件，可追溯绘制过程
3. 使用 AI 检测工具（如 Hive Moderation）辅助鉴别
4. 一旦确认冒充 → 一票否决，列入永久黑名单
5. 如果 CP 事先申明使用 AI 辅助（如 AI 草稿 → 人工精修），则按AIGC 辅助生产规范执行

🔴 案例：CP 核心画师离职导致批量资产品质断崖 高频

🎬 典型场景还原

APM："CP-Y 这个月交付品质突然变差，和之前完全不是一个水平。"

CP-Y："不好意思，之前负责的主力画师离职了，现在换了新人…"

💡 破局思路

1. 合同要求 CP 人员变动必须提前报备（至少 2 周）
2. 核心角色/高优先级资产要求 指定画师，画师变更需甲方同意
3. 对 A/S 级 CP 定期了解其团队稳定性，作为评级维度之一
4. 不把鸡蛋放一个篮子里：同类型资产至少有 2 家 CP 可交付

📌 9. Do / Don't 示例

9.1 供应商准入的正确姿势

✅ Do（正确示范）

• 所有新 CP 必须经过试制评估再正式合作

• 准入评估采用多维度打分（品质/工程/沟通/报价）

• NDA + 知识产权协议在试制前就签署

• 新 CP 默认 B 级，用数据逐步升级

• 维护供应商储备池，避免临时找不到替代

❌ Don't（错误示范）

❌ "朋友推荐的，不用试制了直接合作" → 品质无保障

❌ 只看 Portfolio 不看工程能力 → 好看但不能用

❌ 合同都没签就开始下单 → 安全风险巨大

❌ "反正便宜，先用着看" → 管理成本可能比节省的费用更高

❌ 只有一家 CP 且没有备选 → 被动锁定

9.2 供应商评级的正确姿势

✅ Do（正确示范）

• 评级数据驱动，每个分数可追溯到具体指标

• 评级结果对 CP 透明公开，附改进建议

• 升降级规则提前公示，CP 有预期

• 评级与合作权限/结算条件直接挂钩

• 顶级 CP 有正向激励（份额/结算/冠名）

❌ Don't（错误示范）

❌ 评级靠"感觉"和"关系" → 不公平，好 CP 流失

❌ 只罚不奖 → CP 没有动力提升

❌ 评级后不通知 CP → 人家不知道怎么改进

❌ 评级制度有但从不执行 → 形同虚设

❌ 差 CP 一直不淘汰 → 劣币驱逐良币
