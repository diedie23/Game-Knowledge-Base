🎯 什么是通道打包？

在现代游戏管线中，将多张灰度贴图（AO、Roughness、Metallic 等）打包到一张 RGBA 贴图的 4 个通道中， 可以将 GPU 采样次数从 4 次降低到 1 次，同时减少 VRAM 占用和 Draw Call 数量。这是引擎管线的标准做法。

⚠️ 注意事项

  • 色彩空间：数据贴图（AO、Roughness、Mask）必须设为 Linear，切勿勾选 sRGB；仅 BaseColor / Emissive 等表现贴图使用 sRGB
  • 压缩格式：UE5 推荐使用 BC7（Masks 纹理类型）；Unity 推荐 ASTC 6×6（移动端）或 BC7（PC端）
  • 反转通道：Unity 的 Smoothness = 1.0 - Roughness，若从 UE5 迁移，需在 G 通道勾选反转
  • 空通道处理：未载入的通道默认填充纯白（255）。如果 Shader 预期 0 值，请手动调整默认值
