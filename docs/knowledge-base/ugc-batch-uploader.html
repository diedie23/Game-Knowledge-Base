🎯 工具定位

游戏 UGC 项目的资源上传、替换、管理等操作往往需要在网页端编辑器中逐个手动完成，数量一多，工作量巨大且容易出错。

UGC 批量上传助手是一款全自动化工具箱，通过操作录制 + 智能回放技术，将上传、替换、预加载切换等重复操作变为一键批量执行。v4.0.0 升级为三 Tab 多模块架构，覆盖 UGC 编辑器日常操作全流程。

📑 三大功能模块

Tab 1
📤 批量上传
向导式四步上传：选择窗口 → 录制流程 → 预览验证 → 批量执行。支持自动翻页、失败重试、完成检测。
Tab 2
🔄 资源替换管理
图片搜索定位 + 一键替换/删除。拖入源图 → 感知哈希匹配 → 自动滚动定位 → 执行替换。
Tab 3
⚡ 预加载批量管理
批量选择资源 → 自动排序优化导航路径 → 逐个切换预加载开关，支持关键词筛选和页码范围过滤。

✨ 核心功能

🎯
窗口锁定录制
Win32 API 精准锁定目标窗口，所有坐标转换为窗口内相对坐标，窗口移动不影响回放
📹
操作录制与回放
支持鼠标点击/双击/右键/滚轮、键盘按键/组合键、文件选择对话框等所有操作类型
📸
截图标注验证
每步操作自动截图 + 点击位置红圈标注，回放前可预览全部步骤确保准确
🔍
感知哈希搜索
pHash + OpenCV 模板匹配双引擎，拖入图片即可在编辑器资源列表中精准定位
📊
资源索引系统
全量/增量扫描建立资源索引，记录位置、缩略图、哈希，支持翻页感知和重复检测
🗂️
多项目管理
每个游戏项目独立数据隔离：索引、工作流、日志、缓存互不干扰，一键切换
🔄
失败自动重试
上传完成智能检测，失败自动重试（默认 3 次），确保批量上传零遗漏
📑
工作流保存
录制的操作流程可保存为 JSON 工作流文件，下次直接加载复用，无需重新录制
🧭
智能导航路径
预加载批量操作时，自动按页码排序选中资源，最优路径减少滚动次数

📖 批量上传流程（Tab 1）

1️⃣ 选择目标窗口 — 从已打开的窗口列表中选择 UGC 编辑器窗口
　　→ Win32 API 获取窗口句柄，锁定坐标系

2️⃣ 录制上传流程 — 手动演示"上传一张图"的完整操作（按 F10 停止录制）
　　→ 自动记录每次点击的相对坐标、键盘操作、等待时间
　　→ 每步自动截图，红圈标注点击位置

3️⃣ 预览验证步骤 — 查看录制的每一步操作及截图，确认无误
　　→ 支持删除/编辑/调整步骤

4️⃣ 选择图片 & 批量执行 — 选择待上传的图片文件夹
　　→ 每张图片自动回放录制的操作流程
　　→ 自动翻页、失败重试、进度实时显示
　　→ F6 随时停止 / F7 暂停恢复

🔄 资源替换流程（Tab 2）

1️⃣ 建立资源索引 — 首次使用时扫描编辑器资源列表（支持增量更新）
　　→ 自动截图每一屏，识别资源缩略图，计算感知哈希

2️⃣ 搜索目标资源 — 拖入要替换的图片，或从索引缩略图中浏览选择
　　→ pHash 汉明距离 + OpenCV 模板匹配双重验证

3️⃣ 自动定位 — 工具自动滚动到资源所在页面，高亮显示目标位置

4️⃣ 一键替换/删除 — 自动执行：删除旧资源 → 上传新版本

⚡ 预加载管理流程（Tab 3）

1️⃣ 加载资源索引 — 显示所有资源的缩略图列表

2️⃣ 选择目标资源 — 支持关键词筛选、页码范围过滤、多选

3️⃣ 配置操作位置 — 首次使用时录制预加载开关和返回按钮的坐标

4️⃣ 批量执行 — 自动按最优页码顺序逐个切换预加载开关
　　→ 滚动到目标页 → 点击资源 → 切换开关 → 返回列表 → 下一个

⌨️ 快捷键

F6 停止当前操作（上传/替换/预加载）
F7 暂停 / 恢复执行
F10 停止录制
F5 开始回放（可配置）

⚙️ 可配置参数

参数说明默认值
batch_size每批上传图片数量10
upload_interval_sec每张图片上传间隔（秒）1.5
page_turn_delay_sec翻页后等待时间（秒）2.0
retry_count上传失败重试次数3
auto_page_turn是否自动翻页开启
similarity_threshold图片匹配相似度阈值0.95
scroll_wait滚动后等待截屏时间（秒）1.5

🗂️ 多项目管理

每个游戏项目拥有独立的数据目录，互不干扰：

数据类型说明
资源索引该项目的所有资源缩略图、哈希、位置信息
工作流文件录制保存的操作流程 JSON
操作日志每次上传/替换/预加载的详细日志
项目配置窗口标题关键词、滚动速度、网格参数等
💡 提示：首次使用请先创建项目并设置窗口标题关键词（用于自动匹配 UGC 编辑器窗口）。
💡 提示：如果 exe 文件被杀毒软件拦截，请添加到白名单。这是 PyInstaller 打包的正常现象。工具使用了 Win32 API 和 pyautogui 进行窗口操作，部分杀毒软件可能误报。
