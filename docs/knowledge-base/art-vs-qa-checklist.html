🧑‍🎨 2. 角色走查清单

🔷 2.1 模型质量

3D角色 模型基础质量检查

#🔍 检查项📌 标准☐
1整体比例与原画一致，无变形☐
2面数≤ 红线值 (中端基准)☐
3模型完整性无破面、漏洞、内翻法线☐
4拓扑质量布线合理，变形区域加密☐
5材质球数≤ 规定数量☐
🎨 2.2 贴图质量

贴图 贴图清晰度与规格检查

#🔍 检查项📌 标准☐
1贴图清晰度近景查看不模糊☐
2UV 利用率≥ 75%☐
3贴图尺寸符合红线标准☐
4色差检查与原画色彩一致（引擎内对比）☐
5接缝处理身体部件接缝不可见☐
6sRGB 设置Normal/Mask 未开 sRGB☐
🏃 2.3 动画质量

动画 动作流畅性与穿模检查

#🔍 检查项📌 标准☐
1待机动画自然循环，无抖动☐
2移动动画无滑步，Root Motion 正确☐
3攻击动画打击感明确，帧率流畅☐
4技能动画特效同步，无穿模☐
5过渡自然度状态切换无闪烁/抖动☐
6穿模检查全动画检查无部件穿插☐
7死亡/受击表现明确，不影响游戏☐

特效 技能特效播放与性能检查

🏔️ 3. 场景走查清单

🌅 3.1 视觉质量

场景美术 光影、接缝与整体风格检查

#🔍 检查项📌 标准☐
1整体光影光照方向一致，无异常阴影☐

#🔍 检查项📌 标准☐

2可行走区域导航网格完整，无空洞☐
3传送点/出入口功能正常☐

5LOD 切换切换平滑，无闪烁☐

性能 帧率、面数与内存达标性检查

#🔍 检查项📌 标准☐

🚨 案例：刘海屏适配遮挡

高频 每个有 UI 适配的版本几乎必遇

🎬 典型场景：iPhone 15 Pro 上主界面左上角的返回按钮被刘海遮住了，点不到。程序硬编码 top: 44px 偏移量来避让刘海区，但不同厂商安全区标准不同。

🔍 根因：硬编码偏移值，未使用系统 SafeArea API；不同厂商安全区域标准不同。

💡 破局：① 统一使用 SafeArea API 适配 ② 适配检查矩阵覆盖全部目标机型 ③ 建立设备适配回归清单
