> **编写日期**: 2026-04-20  
> **目标**: 从功能完善度和交互体验两个维度，提出可落地的专业升级路线

## 🚀 升级方案一览

### 第一阶段：交互体验重构（v7.5 · 推荐优先）

#### 1. 实时双视图对比
- **痛点**: 用户调整参数后，只能在单个画布上看效果，无法与原图对比
- **方案**: 新增「原图 ⇆ Mask」分屏对比模式
  - 上下 / 左右分屏切换
  - 拖动分割线实时滑动对比
  - 鼠标悬浮区域放大镜同步两侧
- **交互设计**: 工具栏新增 `👁️ 对比` 按钮，快捷键 `Tab`

#### 2. 参数调节实时预览（WYSIWYG）
- **痛点**: 目前调整容差/饱和度后需要重新执行智能分析才能看到变化
- **方案**: 
  - 滑块拖动时实时更新 Mask 预览（使用 Web Worker 异步计算）
  - 添加「预览品质」档位（Draft/Normal/HQ），Draft 模式下按 1/4 分辨率实时预算
  - 参数变更后自动延迟 200ms 触发预览刷新（防抖）
- **技术要点**: 将色彩匹配逻辑移入 Web Worker，主线程保持 60fps 响应

#### 3. 智能引导式工作流
- **痛点**: 新用户不知道该走哪步，专家用户觉得步骤繁琐
- **方案**: 引入「工作流引擎」概念
  - 新手模式: 只显示 3 步（导入 → 一键分析 → 导出），隐藏所有高级参数
  - 专家模式: 展开全部参数、支持跳步操作
  - 新增「快速模式」：拖入图片 → 自动分析 → 自动生成 → 弹出导出（全程 < 3 秒）
- **交互设计**: 顶部进度条改为可点击的步骤导航，支持前后跳转

#### 4. 操作历史可视化
- **痛点**: 当前撤销/重做是线性栈，用户无法查看和选择历史中间状态
- **方案**: 
  - 新增历史面板，每步操作生成缩略图快照
  - 支持点击任意历史节点回滚
  - 支持分支历史（从中间状态创建新分支）
- **参考**: Photoshop 历史记录面板

### 第二阶段：功能深度扩展（v8.0）

#### 5. 批量处理模式
- **痛点**: 游戏项目中通常需要处理几十上百张角色贴图，逐张操作效率低
- **方案**:
  - 支持拖入文件夹/多选文件批量导入
  - 「模板模式」：在一张图上配置好通道分配方案 → 保存为模板 → 一键应用到所有同类图
  - 批量进度条 + 错误日志
  - 支持后台队列处理（Web Worker Pool）
- **导出**: 批量 ZIP 打包下载，文件名遵循命名模板

#### 6. 局部精修画笔升级
- **痛点**: 现有画笔只有基础涂抹，缺少精修能力
- **方案**:
  - 新增画笔类型：软边笔、硬边笔、橡皮擦、渐变笔
  - 画笔大小/硬度/流量参数
  - 按住 Alt 临时切换为吸色器
  - 支持画笔压感（Pen Tablet / iPad Pencil）
  - 「魔棒选区」工具：点击区域自动扩展选区，仅在选区内操作
- **快捷键**: `B` 画笔 / `E` 橡皮 / `G` 渐变 / `[` `]` 调整大小

#### 7. 通道混合/运算
- **痛点**: 有时需要将两个通道的结果叠加、相减或混合
- **方案**:
  - 通道混合面板：支持 Add / Subtract / Multiply / Screen / Overlay 混合模式
  - 可视化混合预览
  - 混合权重滑块（0~100%）
  - 支持用一个通道作为另一个通道的蒙版

#### 8. AI 辅助色彩分析（可选）
- **痛点**: 智能分析基于传统 K-Means 聚类，对复杂贴图（渐变多、噪点多）效果有限
- **方案**:
  - 集成轻量语义分割模型（如 ONNX Runtime + 预训练 U-Net），自动识别角色各部位
  - 「AI 建议通道分配」：根据语义区域自动推荐 R/G/B/A 分配方案
  - 保留手动覆盖能力
- **注意**: 模型需 < 5MB，推理时间 < 2 秒，降级为传统模式兜底

### 第三阶段：生态与协作（v9.0）

#### 9. PSD 分层导出
- **需求**: 美术同事习惯在 PS 中检查 Mask，希望一键导出 PSD 文件
- **方案**:
  - 使用 [ag-psd](https://github.com/nicktornow/ag-psd) 或类似库生成 PSD
  - 每个通道作为独立图层，图层名 = 通道名
  - 保留原图作为背景图层
  - 支持导出带 Alpha 通道的 PSD

#### 10. 云端工程同步
- **方案**:
  - 工程 JSON 支持上传至云端（腾讯云 COS / 团队共享盘）
  - 生成分享链接，协作者打开即可恢复完整工作状态
  - 版本管理：同一工程的不同版本可对比差异

#### 11. 键盘快捷键体系
- **方案**:

| 快捷键 | 功能 |
|--------|------|
| `Space` + 拖动 | 平移画布 |
| `Ctrl + Z` / `Ctrl + Shift + Z` | 撤销 / 重做 |
| `Ctrl + +` / `Ctrl + -` | 缩放 |
| `Tab` | 切换原图/Mask对比 |
| `1-9` | 快速切换通道 |
| `Ctrl + S` | 保存工程 |
| `Ctrl + E` | 快速导出 |
| `B` / `E` | 画笔 / 橡皮 |
| `[` / `]` | 画笔缩小 / 放大 |
| `Ctrl + Shift + I` | 反选 |
| `F` | 全屏预览 |

#### 12. 国际化 (i18n)
- 当前界面为中文，增加英文/日文支持
- 语言包 JSON 化，UI 文本全部走 `M.L` 字典
- 用户可在设置中切换语言

## 🎨 交互优化专项

### UI 微调建议

| 项目 | 现状 | 建议 |
|------|------|------|
| 颜色吸取反馈 | 放大镜 + 文字提示 | 增加吸取时的「涟漪动画」+ 声音反馈（可关） |
| 通道卡片 | 平铺排列 | 支持拖拽排序，改变 RGBA 顺序 |
| 导出格式选择 | 按钮散布 | 改为统一导出弹窗，一站式选格式/命名/尺寸 |
| 参数面板 | 固定在右侧 | 可折叠/可拖动/可固定，适配不同屏幕 |
| 空状态引导 | 仅文字 | 增加动画示意图，展示「拖放文件」手势 |
| 移动端适配 | 基本响应式 | 针对 iPad 优化触控区域大小、手势操作 |

### 操作流程优化

```
当前流程（7步）:
导入 → 预处理 → 吸色 → 分配通道 → 调参 → 预览 → 导出

建议快速流程（3步）:
导入 → 一键智能分析 → 预览并导出
        ↓（不满意时）
      手动精修模式（展开全部参数）
```

### 错误恢复与容错

- 导入失败时提供明确的错误原因 + 解决建议
- 操作超时时自动保存工程 JSON 到 localStorage
- 浏览器崩溃恢复：重新打开页面时自动检测并恢复未保存工程
- 所有弹窗支持 `Esc` 关闭
