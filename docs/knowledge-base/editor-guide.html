🎉 不用写代码 · 所见即所得 · 打开浏览器就能编辑
🎯 一句话总结：用浏览器打开任意知识库文档 → 点右上角「✏️ 编辑模式」→ 直接改文字 → 点「保存」下载文件 → 替换原文件。就这么简单！

🔰 基础操作：5 步完成文档修改

1 打开文档

在文件管理器中找到要修改的 .html 文件，双击打开（默认会在浏览器中打开）。

📍 文档位置：H:\游戏项目知识库\docs\knowledge-base\

2 进入编辑模式

页面右上角会出现一个 紫色渐变按钮「✏️ 编辑模式」，点击它。

此时页面顶部会出现提示：📝 编辑模式已开启，并且可编辑区域会显示蓝色虚线边框。

3 修改内容

改文字：直接点击任何有蓝色虚线边框的区域，光标会出现，像在 Word 里一样打字修改。

换图片：双击任何图片，会弹出文件选择框，选择新图片即可替换。

改表格：直接点击表格单元格修改数据。

⚠️ 注意：不要尝试删除整个卡片、章节等排版区域 — 系统会自动保护它们不被删除，确保文档结构不会被破坏。你只能修改文字和图片内容。
4 保存文档

修改完成后，右上角工具栏有两个保存按钮：

按钮作用使用场景
💾 保存并覆盖 下载一个与当前文件同名的 .html 文件 修改旧文档时使用
📄 另存为新文档 弹出输入框，自定义文件名后下载 基于模板创建新文档时使用
5 替换原文件

下载的文件默认在你的「下载」文件夹中。把它复制到文档目录并替换原文件即可：

📍 目标路径：H:\游戏项目知识库\docs\knowledge-base\

💡 提示：如果你有 Git 操作权限，替换文件后记得 git add → git commit → git push。如果不会操作 Git，把文件发给知识库管理员即可。

📋 两大常用工作流

🔧 场景 A：修改一篇旧文档（如更新数据、修正错误）
  1. 在文件管理器中双击打开那篇 .html 文件
  2. 点右上角「✏️ 编辑模式」
  3. 直接在页面上修改文字内容 / 表格数据
  4. 点「💾 保存并覆盖」— 浏览器会下载一个同名文件
  5. 把下载的文件复制到原目录，覆盖原文件
  6. 完成！ ✅
📝 场景 B：基于模板创建一篇全新文档
  1. 双击打开 Blank_Template.html（空白模板）
  2. 点右上角「✏️ 编辑模式」
  3. 修改标题、正文、表格等所有占位内容为你的实际内容
  4. 点「📄 另存为新文档」
  5. 输入文件名（如 11月美术复盘），点确认
  6. 浏览器会下载 11月美术复盘.html
  7. 把文件放到 knowledge-base 目录即可
  8. 完成！ ✅ 模板文件不会被修改，下次还能继续用
📑 场景 C：复制一篇旧文档的排版，修改为新内容
  1. 找到一篇排版你喜欢的现有文档，双击打开
  2. 点右上角「✏️ 编辑模式」
  3. 把文字内容改成你的新内容
  4. 点「📄 另存为新文档」，输入新文件名
  5. 下载后放到 knowledge-base 目录
  6. 原文档不会受影响，因为你没有覆盖它 ✅

❓ 常见问题

Q：我不小心删掉了一大段内容怎么办？

别慌！有两个方法：

  • 方法一：按 Ctrl + Z 撤销
  • 方法二：点击工具栏的「🗑️ 放弃修改」按钮，会刷新页面恢复到修改前的状态
Q：为什么有些区域我删不掉？

这是 防破坏保护机制。文档的排版骨架（如章节容器、卡片外框、表格行等）被锁定了，防止误操作破坏文档结构。你可以修改里面的文字内容，但不能删除整个结构块。

Q：我想增加一个新章节怎么办？

目前可视化编辑器主要支持 修改已有内容。如果需要增加新章节或复杂排版，建议：

  • 找到一篇有类似结构的文档作为参考
  • 联系知识库管理员帮忙添加新的章节结构
  • 或者让管理员提供一个包含更多章节的模板
Q：下载的文件在哪里？

和你平时浏览器下载的文件位置一样，通常在 C:\Users\你的用户名\Downloads 文件夹中。

Q：我看不到右上角的编辑按钮？

请确认文档末尾是否包含这行代码：<script src="editor-kit.js"></script>。如果没有，说明该文档还未接入编辑器，请联系管理员。

🔧 管理员指南（给维护知识库的同学看）

如何为新文档接入编辑器

只需在任意 .html 文档的 </body> 标签前加一行：

<script src="editor-kit.js"></script>
💡 注意：editor-kit.js 文件需要和文档在同一目录下（即 knowledge-base/ 目录）。如果文档在子目录中，路径需要相应调整，如 <script src="../editor-kit.js"></script>。

如何制作自定义模板

  1. 复制一份 Blank_Template.html 或任意现有文档
  2. 修改标题为模板名（如「月度复盘模板」）
  3. 把正文内容改为占位提示（如"在此输入复盘总结…"）
  4. 确保文件末尾有 <script src="editor-kit.js"></script>
  5. 保存即可。团队成员打开后「另存为新文档」就能基于此模板创建
