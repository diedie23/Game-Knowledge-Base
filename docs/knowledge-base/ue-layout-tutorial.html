绘制为 (Draw As) —— 九宫格核心！
如果是带圆角的弹窗底板，拉伸会变形。必须将 Draw As 设为 盒体 (Box)，并将 Margin 设为图片圆角的安全百分比（如 0.25，代表保护四周 25% 不被拉伸）。

A (Alpha) 不透明度：0 表示 0%（全透明），1 表示 100%（全实色）。

【即时预览】：调节颜色、尺寸和九宫格参数时，视口中心会立刻实时刷新变化。

🔤 3. 文本双雄（LGText 与 Rich Text）对比详解

🔤 LGText (项目定制文本)

替代 UE 默认 Text 的首选。做了本地化多语言包装和性能优化。

  • 面板核心参数：
    • Text (文本)：填入你要显示的默认字。
    • Font (字体)：选择字库及 Size (字号大小)。
    • Color (颜色)：改变字体颜色。
    • Shadow Offset：字体的阴影偏移量（X/Y 设为 1 则有右下角阴影）。
    • Justification (对齐)：居左、居中、居右对齐。
  • 适用场景： 标题、按钮文案、单色属性数值。

📝 Rich Text Block (富文本)

通过 HTML 标签驱动的图文混排神器。

  • 面板核心参数：
    • Text Style Set：必须！要在这里挂载一个数据表，定义好红色标签、黄色标签。
    • Text (文本)：支持写标签，例如：
      你获得了 <Red>100</> 点伤害！
      花费 <img id="coin"/> 500 购买。
  • 适用场景： 聊天频道（带表情）、多色彩的物品掉落提示、技能描述（重点字加粗标红）。

【即时预览】：无论哪种文本，修改字号、颜色、打出富文本标签后，编辑器均会100%立刻渲染刷新。

Percent / Value (当前数值)：核心参数！取值范围固定在 0.0 ~ 1.0 之间（代表 0% 到 100%）。比如填入 0.65，进度条就会只满 65%。修改此值，画面立刻刷新。

滑条专属特性：滑条比进度条多一个 Thumb Image (滑块图标)，就是玩家按住拖动的那个按钮。

【避坑注意】：如果在程序中血量是 500 / 1000，必须在代码里除一下变成 0.5 传给 UI，不能直接把 500 传给进度条！

使用场景：当一个控件（比如 Button）放在 Horizontal Box 里时，它失去了调节大小的权利，只能被动拉伸。如果我想强行让这个按钮就是 120x50 像素怎么办？

解法：在 Button 外面包裹一层 Size Box (尺寸框)，并勾选 Width/Height Override。尺寸框就像一个钢铁牢笼，强行把里面的控件框死在设定好的尺寸里。修改后立刻生效。

5.2 Widget Switcher (控件切换器) —— 抽屉式翻页

▼ 切换器 (Switcher)
Active Widget Index
1
➔

核心特性：你可以把十个界面板子塞进切换器里，但它同一时间永远只显示一个！

Active Widget Index (激活索引)：0 代表显示第一个子节点，1 代表显示第二个节点。在编辑器中修改数字，能立刻看到翻页效果。

使用场景：商城顶部的“武器/防具/消耗品”页签切换。点页签时让代码改变这个 Index 即可瞬间切面板，无需创建和销毁，性能优秀。

5.3 水平框 (H-Box) / 垂直框 (V-Box) / 安全区 (Safe Zone) 对比

⏸️ 水平/垂直框

排队神器。把控件丢进去，它们就会自动向右或向下排列。
核心参数：在子节点的插槽中选择 Auto(按内容尺寸) 还是 Fill(填满剩余全部空间)。通过 Padding 控制兄弟间的间隙。

🛡️ Safe Zone (安全区)

全面屏守护者。勾选 Pad Left/Right/Top/Bottom。
核心场景：移动端设备有“刘海”、挖孔、圆角。把整个界面放在 Safe Zone 里，系统会自动把 UI 往里挤，防止按钮被屏幕黑边遮挡。

8.2 对齐 (Alignment) —— 绝对居中的密码

当你要把弹窗绝对居中时，光把锚点设为“居中”是不够的（那只是以控件的左上角点对齐屏幕中心）。
必须将 Alignment 设为 X: 0.5, Y: 0.5。这代表用控件自己的正中心点，去贴合锚点。这是所有弹窗类 UI 必须遵守的铁律！

8.3 多端与微信小游戏适配专属规范

📱 微信小游戏/手游端必须遵循的布局层级树：

由于微信小游戏在各种带鱼屏手机（如 21:9）、异形屏上有严重差异，且右上角必定有一个不可去掉的“胶囊按钮（更多/关闭）”，如果乱摆放会导致按键完全点不到。

标准根节点搭建规范范例：

[根容器] Canvas Panel (画板)
 ├── 🖼️ Image [背景大图] 
 │    └─ 规范：锚点选「四边全屏拉伸(大方块)」，Offsets全部归0。无视刘海直接铺满全屏，做视觉延伸。
 ├── 🛡️ Safe Zone [安全区] 
 │    ├─ 规范：紧接背景之上，勾选四边Pad，系统会自动挤开微信胶囊和刘海屏幕的黑边。
 │    └─ 🥞 Overlay [全屏覆层] (放在SafeZone里铺满)
 │         ├── 📦 VBox/HBox (各种功能面板内容，绝对安全)
 │         └── 👆 Button [退出/返回按钮] 
 │              └─ 规范：锚点设为右上角。因为有SafeZone保护，绝对不会和微信的胶囊按钮重叠！
