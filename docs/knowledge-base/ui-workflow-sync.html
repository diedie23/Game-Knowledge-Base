📑 3. 各阶段 · 角色 · Input / Output

① 策划输出需求文档

📥 Input
• 玩法设计文档 • 功能原型/草图
• 参考竞品 • 系统数据结构
📤 Output
• UI 需求文档（含页面清单）
• 功能逻辑说明 • 数据字段定义
责任人策划预计1-2 天

② 需求评审 NEW

📌 触发：策划完成需求文档后、进入交互设计之前。 👤 主导：策划　 ✅ 决策：项目经理/制作人
📥 Input
• 策划需求文档 • 技术可行性预评估
📤 Output
• 评审纪要 • 需求冻结确认 • 排期承诺
主导策划参与交互UILayout客户端预计0.5 天
🔴 评审通过后需求冻结，变更需走正式变更流程。

③ 交互设计

📥 Input
• 已冻结的需求文档 • 平台设计规范
📤 Output
• 交互线框图 • 页面跳转逻辑图 • 异常状态说明
责任人交互设计师预计3-5 天

④ 交互评审 → ⑤ 交互图交付

📥 评审输入
• 交互稿 • 需求文档（对照）
📤 交付输出
• 正式交互标注文档 • 同步至蓝湖/Figma
参与策划交互UI客户端预计0.5+0.5 天

⑥ UI 设计

📥 Input
• 交互标注文档 • 视觉风格指南 • UI 组件库
📤 Output
• 高保真 UI 设计稿 • 标注 • 图标素材
责任人UI 设计师预计5-7 天

⑦ UI 评审 → ⑧ UI 图交付

📥 评审输入
• UI 设计稿 • 交互稿（对照）
📤 交付输出
• 切图（PNG/SVG，偶数尺寸）• 标注 • 9宫格
参与策划交互UILayout预计0.5 天评审 + 1 天切图

⑨ 资产合规检查 NEW

📥 Input
• UI 切图资产包 • 命名规范文档
📤 Output
• 合规报告（PASS/FAIL）• 不合规项清单
责任人UILayout预计自动化 30 秒
🔴 门禁：模块_功能_状态_尺寸、偶数尺寸、PNG-24/WebP、sRGB、9宫格完整。

⑩ Layout 制作（白模 + 切正式 + 正式蓝图）

📥 Input
• 交互标注文档 • 合规切图 • 引擎 UI 规范
📤 Output
• 功能蓝图（白模）• 正式资产替换 • 正式蓝图
责任人Layout 工程师预计3-5 天
💡 Layout 白模可与 UI 设计并行（交互交付后即启动），客户端可基于白模并行开发。

⑪ VX 动效（预演 + 正式制作）

📥 Input
• 交互动效说明 • UI 设计稿 • 引擎动效能力
📤 Output
• 预演视频/GIF • 正式动效资产 • 参数文档
责任人VX 动效设计师预计2-4 天

⑫ 动效评审 NEW

📌 触发：仅含动效需求时触发，无动效可跳过。 👤 主导：VX　 ✅ 决策：交互 + 客户端
📥 Input
• 动效预演视频 • 参数文档 • 性能预算
📤 Output
• 结论（✅通过 / 🔧调整 / ❌砍除）
• 性能评估
主导VX决策交互客户端可选UI预计≤0.5 天
⚠️ 性能优先于动效。≤30分钟当场出结论。

⑬ 客户端集成 → ⑭ 界面走查 → ⑮ 交付

📥 集成输入
• 正式蓝图 • 动效资产 • 功能逻辑+接口文档
📤 走查/交付输出
• 走查通过 → 资产归档 + 上线
• 问题清单 → 分流至责任人
集成客户端
走查主导策划UILayout功能测试QA协助交互VX预计3-5 天集成 + 2-3 天走查

✅ 4. 关键评审节点与通过标准

评审节点参与人通过标准打回条件上限
②需求评审 NEW策划主导，全员参与需求完整无歧义、技术可行、排期合理需求模糊、技术不可行2次
④交互评审策划+交互+UI+客户端逻辑完整、交互流畅、异常已处理遗漏分支、路径过深2次
⑦UI评审策划+交互+UI+Layout风格统一、标注准确、适配完整风格偏差、标注缺失2次
⑨资产合规 NEWUI+Layout命名100%合规、尺寸/格式通过、sRGB自动化检查FAIL不限
⑫动效评审 NEWVX主导，交互+客户端决策动效与设计意图一致、性能在预算内DrawCall超预算/表现不达标2次（可砍除）
⑭界面走查策划/UI/Layout主导，QA功能像素级一致、功能正确、动效流畅P0/P1问题存在3次

🔴 铁律：超过最大迭代次数仍未通过 → 上升至项目经理介入。

🔀 5. 走查问题分流

问题类型典型表现分流至SLA
🖌️ UI问题颜色偏差、图标错误、字体不一致UI设计师1天
📐 Layout问题蓝图错位、适配异常、层级错乱Layout0.5天
💻 开发问题功能逻辑错误、数据异常客户端1-2天
🎨 交互问题流程不通、反馈缺失交互1天
✨ 动效问题动效卡顿、方向/时长不合理VX1天
✅ 闭环：发现 → 填写修改清单 → 分流 → 修复 → 自测 → 复查 → 验收关闭。

⏱️ 6. 参考工期速查

阶段环节参考天数责任角色
阶段一①②策划+评审2-3 天策划
阶段一③④⑤交互3-5 天交互
阶段二⑥⑦⑧⑨UI5-8 天UI
阶段二⑩Layout3-5 天Layout
阶段三⑪⑫动效2-4 天VX
