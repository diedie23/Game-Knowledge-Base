📋 Tab 1：资源检测

检测维度
检测项说明规则示例
尺寸超限图片宽高超过设定最大尺寸最大 2048px
尺寸规则宽高不符合 2 的幂/倍数规则2的幂次方 / 4的倍数
命名规范文件名不匹配正则表达式^[A-Za-z0-9_]+$
文件大小文件体积超过设定上限最大 500KB
重复资源MD5 + 多哈希初筛 + 8 层验证跨尺寸重复检测
使用流程
1️⃣ 设置检测规则（最大尺寸、尺寸规则、命名规则、最大文件大小、包含格式）
2️⃣ 添加扫描文件夹（按钮添加 / 拖拽添加，支持多个）
3️⃣ 点击【开始检测】（支持暂停/继续/取消）
4️⃣ 查看检测结果（颜色区分：🟢通过 🔴问题 🟠重复 ⚪忽略）
5️⃣ 后续操作：右键菜单 / 导出报告（Excel/CSV/JSON/HTML）/ 图表统计

🔢 Tab 2：图片续排

使用流程
1️⃣ 选择目标文件夹（自动扫描已有图片，分析命名格式）
2️⃣ 确认/修改命名设置（前缀、数字位数）
3️⃣ 选择处理方式（移动文件 / 复制文件）
4️⃣ 拖拽待处理图片到拖拽区
5️⃣ 点击【开始续排】→ 自动从最大序号+1开始编号
💡 示例：目标文件夹已有 hero_001.png ~ hero_042.png，拖入5张新图片 → 自动生成 hero_043.png ~ hero_047.png
