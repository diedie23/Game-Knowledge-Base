5.2 关键协作节点

✅ 正确协作 — 原画阶段就拉动效师评审

在线稿阶段让动效师看一眼结构可行性。例如："这个飘带能不能做骨骼？""这个发型旋转会不会露出空白？"提前发现问题，避免完稿后才返工。

❌ 错误协作 — 原画完稿后甩图给动效

设计稿画完了才交给动效师，结果发现："手臂和身体重叠区域没预留""这个配饰结构做不了旋转"。原画返工 3 天。

⑨ PM 排期模型与甘特图模板

📌 PM 核心痛点："角色什么时候能做完？""并行几个角色不会阻塞？""哪个阶段经常延期？"——这些问题需要量化的排期模型，而不是靠感觉估时间。
9.1 单角色排期基准（人天）
阶段最小值标准值上限值并行度前置依赖延期风险
P1 需求定义0.25d0.5d1d可并行—低
P2 原画设计2d3d5d独占1人P1高（返修多）
P3 PSD 拆分0.25d0.5d1d可外包P2低
P4 骨骼绑定0.5d1d2d独占1人P3中
P5 动画制作2d4d7d独占1人P4高（连招调试）
P6 换色/换装0.5d1d2d可并行P5中
P7 TA 审查0.25d0.5d1d可并行P5/P6低（自动化后）
P8 导出打包—CI 自动0.5d自动P7极低
P9 引擎集成+联调0.5d1d2d独占1人P8中
📊 关键路径：P1 → P2 → P3 → P4 → P5 → P7 → P8 → P9，标准关键路径 = 10.5 人天。P6（换色/换装）可与 P5 后半段并行。PM 排期必须加 20% 缓冲系数（= 12.5 人天 ≈ 2.5 周）。
